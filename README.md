# install irc client
```
sudo apt-get install irssi
```

# start irssi (irc client)
```
irssi
```

# commands in irssi
```
/connect irc.freenode.net 8001
/nick ???NICKNAME???
/join #h15
```

# misc commands
```
/part --> chat verlassen
```

# alias
alias irssi="irssi -c irc.freenode.net 8001 -n ???NICKNAME???"
